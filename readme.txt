VUT_FIT_IJA_project

Projekt do předmětu IJA - Aplikace pro zobrazení a editaci diagramů tříd a sekvenčního diagramu.

Authors
Marek Roman - xmarek74 (Team leader)
Václavek Lukáš - xvacla32

Program je tvořen pomocí Java 11.
Pro grafické uživatelské rozhraní je využit JavaFX.
Pro překlad a spuštění je využit nástroj Maven.
